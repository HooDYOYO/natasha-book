КНИГА ДЛЯ НАТАШИ — версия 2

1. Скопируй все экспортированные PNG в папку pages.
2. Названия должны остаться такими же:
   A4 - 2.png
   A4 - 3.png
   ...
   A4 - 42.png
3. Открой index.html двойным кликом.

Обложка:
- положи готовую картинку обложки в assets;
- назови её cover.png.
Если cover.png нет, сайт покажет временную текстовую обложку.

Управление:
- кнопки ← →;
- стрелки клавиатуры;
- пробел — вперёд;
- нажатие по краям книги;
- свайп на телефоне;
- кнопка ⛶ — полноэкранный режим;
- кнопка ⌂ — вернуться к обложке.

Важно:
Сейчас развороты используются как цельные PNG. Отдельное падение фото и построчное появление текста добавляется позже для выбранных страниц, не ломая эту версию.
